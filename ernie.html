Chapter 1: FREQUENCY DOMAIN ANALYSIS

1.1 Introduction

RC-Network Example

\[ \frac{V_{0}(s)}{V_{i}(s)} = \frac{1/sC}{R+1/sC} = \frac{1}{\tau s+1}, \quad \tau = RC \]
\[ v_{i}(t) = 2\sin(\omega t + 30^\circ) \]
  • \( R = 1k\Omega \)
  • \( C = 10^{-4}F \)
  • \( \tau = 0.1\ \text{sec} \)

General Frequency Response

\[ r(t) = A \sin(\omega t) \]
\[ R(s) = \frac{A \omega}{s^2 + \omega^2} \]
\[ Y(s) = G(s)R(s) \]
\[ y_{ss}(t) = A |G(j\omega)| \sin(\omega t + \angle G(j\omega)) \]
  • Amplitude: \( A_o = |G(j\omega)| A \)
  • Phase: \( \phi_o = \angle G(j\omega) + \phi_i \)

Example 1.1: RC-Network

\[ G(s) = \frac{1}{0.1s + 1} \]

Input: \( r(t) = 2\sin(\omega t + 30^\circ) \)

For \( \omega = 5 \) rad/s:

\[ |G(j5)| = 0.8944 \]
\[ \phi(5) = -26.6^\circ \]
\[ y_{ss}(t) = 2 \cdot 0.8944 \sin(5t + 30^\circ - 26.6^\circ ) \]

For \( \omega = 50 \) rad/s:

\[ |G(j50)| = 0.1961 \]
\[ \phi(50) = -78.7^\circ \]
\[ y_{ss}(t) = 2 \cdot 0.1961 \sin(50t + 30^\circ - 78.7^\circ ) \]
